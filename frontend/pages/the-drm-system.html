The DRM System

โครงสร้างสัญญาณของ DRM ประกอบด้วยช่องสัญญาณสำคัญ 3 ส่วน ได้แก่ FAC สำหรับระบุสถานีและช่วยให้เครื่องรับสแกนหาบริการได้รวดเร็ว, SDC สำหรับส่งข้อมูลกำกับและเมทาดาทา เช่น ภาษา ประเภทเนื้อหา และความถี่ทางเลือก, และ MDC สำหรับส่งข้อมูลเสียงและบริการข้อมูลหลัก ระบบยังสามารถรองรับหลายสตรีมภายในช่องสัญญาณเดียว ช่วยให้สามารถให้บริการทั้งเสียงและข้อมูลดิจิทัลร่วมกันได้อย่างมีประสิทธิภาพ

ส่วนประกอบหลักของระบบ DRM

  • FAC (Fast Access Channel) - ช่องเข้าถึงแบบรวดเร็ว ใช้สำหรับระบุตัวตนของสถานีและให้เครื่องรับสามารถสแกนหาบริการได้อย่างรวดเร็ว
  • SDC (Service Description Channel) - ช่องอธิบายบริการ ใช้ส่งข้อมูลกำกับและเมทาดาทา เช่น ชื่อสถานี ภาษา ประเภทเนื้อหา และความถี่ทางเลือก
  • MDC (Main Service Channel) - ช่องบริการหลัก ใช้ส่งข้อมูลเสียงและบริการข้อมูลหลัก

ความสามารถของระบบ

ระบบ DRM มีความสามารถพิเศษดังต่อไปนี้:

  • รองรับหลายสตรีมภายในช่องสัญญาณเดียว
  • ให้บริการทั้งเสียงและข้อมูลดิจิทัลร่วมกันได้อย่างมีประสิทธิภาพ
  • มีความทนทานต่อสัญญาณรบกวนและการเฟดดิ้งสูง
  • รองรับการส่งข้อมูลเพิ่มเติมหลากหลายประเภท
